# Redux Shopping Cart


**Shopping Cart Project** built with **React** + **Redux-Toolkit**.

<br/>

## Features:

- #### Add to Cart
- #### Remove from Cart
- #### Increment / Decrement the Quantity

<br/>

## Technologies used:

- #### **React JS**
- #### **Redux Toolkit**
- #### **SASS**
